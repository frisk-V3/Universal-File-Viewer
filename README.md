# 🗂 Universal File Viewer
音声 / 画像 / 動画 / PDF / HTML 実行 / コード色分け  
ブラウザだけでほぼ全種類のファイルを開ける多機能ビューア。  
※ exe は安全のため非対応。

---

## ✨ Features

- 画像表示（PNG / JPG / GIF / WebP など）
- 音声再生（MP3 / WAV / OGG）
- 動画再生（MP4 / WebM）
- PDF 表示
- HTML 実行（iframe でそのまま動く）
- コードファイルは Prism.js によるシンタックスハイライト
- テキストファイルも読み込み可能
- 複数ファイル同時読み込み対応
- 完全ローカル動作（サーバー不要）

---

## 📁 対応ファイル一覧

| 種類 | 内容 |
|------|------|
| 画像 | 表示 |
| 音声 | 再生 |
| 動画 | 再生 |
| PDF | iframe 表示 |
| HTML | 実行可能（サンドボックス） |
| コード | 色分け表示（Prism.js） |
| テキスト | そのまま表示 |
| exe | ❌ 非対応 |

---

## 🧪 使い方

1. `file-viewer.html` をブラウザで開く  
2. 「ファイルを選択」から任意のファイルを読み込む  
3. 種類に応じて自動で最適なビューアが表示される

## 🖥 HTML 実行について

HTML ファイルは iframe に読み込まれ、  
**そのままブラウザ上で動作します。**

- Web アプリ  
- UI テスト  
- ゲーム  
- ライブラリの動作確認  

などに利用可能。

---

## 🔒 セキュリティ

- exe は読み込まず拒否  
- HTML は iframe 内でサンドボックス実行  
- 完全ローカルで動作するためファイルは外部に送信されない
